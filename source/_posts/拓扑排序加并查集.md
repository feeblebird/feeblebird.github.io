# Chess Tournament

#### 题目大意

* 给出 *N* 个选手的 *M* 种实力大小关系，问是否矛盾。N(2 $\leq$ N $\leq$ 50000)，M(1 $\leq$ M $\leq$ 250000)

#### 解题思路

* 若A < B，连一条从B到A的边
* 若A > B，连一条从A到B的边
* 若A = B，将A和B缩为一点
* 将M种关系放到有向图上，判断图种是否有环存在。缩点用并查集实现，判环可以用拓扑排序实现。

#### 代码

```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int main()
{
    int n,m;
    int k,l;
    scanf("%d%d",&n,&m);
    char c;
    for(int i = 1;i <= m;i ++)
    {
        scanf("%d",&k);
        c = getchar();
        c = getchar();
        scanf("%d",&l);
    }
    return 0;
}

```
